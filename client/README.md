# Commad Line Interface for docshub
---
## Instalation
```
poetry install
```
---
## How to usage:

```
python hseclient/cli.py
```
```
poetry run client
```
---
## Command:
---
```
>>help
```
>выводит список доступных команд
---
```
>>generate
```
>создает пару ключей (публичный;приватный) и csr

Требуется ввод: 
1. Код страны в международном формате 
2. Номер телефона
3. ФИО 
---
```
>>register
```
> Регистрирует пользователя в системе

Требуется ввод: 
1. ФИО
2. Номер телефона
3. Код из звонка
---
```
>>revoke
```
> Удаления пользователя из системы

Требуется ввод: 
1. Код из звонка
---
```
>>login
```
> Аутентификация пользователя в системе

---
```
>>upload
```
> Загрузка файла на сервер

Требуется ввод:
1. Путь к файлу
---
```
>>my_files
```
> Список загруженных файлов и есть ли доступ к ним
---
```
>>sign
```
> Подписывание файла с помошью ЭЦП

Требуется ввод:
1. Имя файла
---
```
>>download
```
> Скачивание файла

Требуется ввод:
1. ID файла
---
```
>>delete
```
> Удаление файла

Требуется ввод:
1. Имя файла
---
```
>>check_available
```
> Проверяет доступность новых документов
---
```
>>send
```
>  Отправляет документ по номеру телефона

Требуется ввод:
1. Имя файла
2. Номер получателя
